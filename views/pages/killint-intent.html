Table Top Card Game

Killing intent is an original game created with the intent of creating a guessing game that will have factions. It was a game concept that I created, overviewed, managed, and developed with the help of my team. The game is a table top card game that has you guessing the other players' cards and trying to eliminate them. The game is designed to be played with 4-8 players and has a playtime of around 30-60 minutes.
